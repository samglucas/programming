# HTML

## Elements
### Headings
```
<h1>: The largest and most important heading. Typically used for the main title of a page.
<h2>: Used for major sections of content.
<h3>: Used for subsections or detailed topics.
<h4>: Used for minor sections or grouped details.
<h5>: Used for specific details.
<h6>: The smallest heading level. Used for fine-grained subtopics or footnotes.
```

```
<p>: Paragraph element, used for regular text.
```
